## jQuery无new构建实例
无new化构建:

$ 就是jQuery的别称

$() 就是在创建jQuery的实例对象

举个简单的例子：
```js
(function(root){
    var jQuery = function() {
        return new jQuery.prototype.init();
    }
    root.$ = root.jQuery = jQuery;
})(this);
```
使用自执行函数将$符挂载到Window对象上
